# Description
This is a PHP CLI package which provides some helpful functions that make creating PHP-related files easier and faster, whilst also maintaining strict visual file data continuity.

# Dependencies
	- php ^5.5
	- symfony/console ^3.2
	- symfony/filesystem ^3.2
